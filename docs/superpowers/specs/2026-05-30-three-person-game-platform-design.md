# 三人行 - 三人联机游戏平台设计规格

**项目名称**: 三人行 (Three's Company)
**版本**: 1.0.0
**创建日期**: 2026-05-30
**状态**: 设计阶段

---

## 1. 项目概述

### 1.1 项目目标

构建一个创新的三人联机游戏平台，提供多种游戏类型和模式，支持手机、平板和电脑全平台访问。

### 1.2 核心价值

- 🎮 **多样性**: 融合4大类型游戏，4种游戏模式
- 👥 **社交性**: 三人联机，实时互动
- 📱 **跨平台**: 完全响应式设计
- 🎯 **易上手**: 直观的游戏规则和界面

## 4. 功能模块设计

### 4.1 首页模块

**功能**:
- 4种游戏模式展示卡片
- 快速开始按钮
- 最新战绩展示
- 玩家设置入口

**组件**:
- `ModeCard`: 模式选择卡片
- `QuickStart`: 快速开始按钮
- `RecentScores`: 最近战绩
- `SettingsModal`: 设置弹窗

### 4.2 房间系统

**功能**:
- 创建房间 (生成6位房间号)
- 加入房间 (输入房间号)
- 玩家准备状态
- 房主控制权

**数据流**:
```
创建房间 → 生成房间号 → 等待加入 → 玩家列表 → 开始游戏
加入房间 → 输入房间号 → 验证 → 加入房间 → 等待开始
```

### 4.3 游戏核心引擎

**游戏状态管理**:
```javascript
GameState = {
  phase: 'waiting' | 'playing' | 'paused' | 'finished',
  currentRound: number,
  totalRounds: number,
  scores: { player1: 0, player2: 0, player3: 0 },
  timer: number,
  settings: GameSettings
}
```

**事件系统**:
- `game:start` - 游戏开始
- `game:round` - 新回合
- `game:end` - 游戏结束
- `player:action` - 玩家操作
- `player:score` - 分数更新

## 5. 游戏类型详细设计

### 5.1 协作挑战类 (A)

#### 5.1.1 你画我猜接龙 🎨

**规则**:
1. 玩家1看到关键词，开始画画 (限时60秒)
2. 玩家2在玩家1画的基础上继续画 (限时60秒)
3. 玩家3根据最终画作猜测答案
4. 猜对得30分，答错得10分鼓励分

**界面**:
- 画布区域 (Canvas)
- 工具栏 (画笔颜色/粗细/橡皮擦/撤销)
- 计时器
- 玩家指示器 (谁在画)

**交互**:
- 触摸/鼠标绘画
- 双指缩放画布
- 颜色选择器
- 撤销上一步

#### 5.1.2 协同解密 🧩

**规则**:
1. 系统生成一个谜题，分为3个线索
2. 每个玩家只能看到1个线索
3. 玩家需要讨论交流线索
4. 在限定时间内共同给出答案 (限时90秒)
5. 答对得50分，答错不得分

**界面**:
- 线索展示区 (各自的线索不同颜色标记)
- 讨论聊天区
- 答案输入框
- 倒计时器

#### 5.1.3 心声传递 💬

**规则**:
1. 玩家1看到一个秘密词汇
2. 玩家1用一句话描述 (不能包含原词)
3. 玩家2根据描述再用自己的话传递
4. 玩家3根据第二个描述猜测
5. 越接近原始答案得分越高

**界面**:
- 词汇展示区 (仅玩家1可见)
- 传递信息展示
- 答案输入区
- 相似度评分

#### 5.1.4 节奏同步 🎵

**规则**:
1. 系统播放一段节奏
2. 三人需要同时点击正确的节奏
3. 系统检测三人点击的时间差
4. 时间差越小分数越高

**界面**:
- 节拍指示器
- 三个玩家的点击区域
- 时间差显示
- 平均得分

### 5.2 竞技对抗类 (B)

#### 5.2.1 知识抢答 🧠

**规则**:
1. 系统展示问题
2. 三人同时可以抢答
3. 第一个点击的人有3秒答题时间
4. 答对得20分，答错-10分
5. 连续答对有连击加分

**界面**:
- 问题区
- 抢答按钮 (三人各一个)
- 倒计时
- 连击显示
- 分数板

#### 5.2.2 反应挑战 ⚡

**规则**:
1. 屏幕显示"准备"文字
2. 随机1-5秒后变为"GO!"
3. 第一个点击的人得分
4. 抢跑(在GO前点击)扣分
5. 10轮后结算

**界面**:
- 状态显示区 (准备/GO!)
- 三个玩家的反应区
- 轮次指示器
- 最短反应时间显示

#### 5.2.3 精准投掷 🎯

**规则**:
1. 显示目标区域
2. 每人3次投掷机会
3. 击中目标中心得10分
4. 偏离中心按距离递减
5. 统计总分

**界面**:
- 目标区域 (Canvas绘制)
- 投掷轨迹
- 分数显示
- 命中记录

#### 5.2.4 文字游戏 🔤

**规则** (成语接龙模式):
1. 系统给出起始成语
2. 每人依次接龙 (15秒思考时间)
3. 回答正确得10分
4. 答错/超时扣5分
5. 连续接上10个成语

**界面**:
- 当前成语显示
- 计时器
- 历史接龙记录
- 下一位玩家提示

### 5.3 创意表达类 (C)

#### 5.3.1 故事接龙 ✍️

**规则**:
1. 系统给出故事开头
2. 玩家1续写一段 (50字内)
3. 玩家2继续
4. 玩家3收尾
5. 最终生成完整故事
6. 每人为他人评分

**界面**:
- 故事展示区
- 编辑输入框
- 字数限制显示
- 评分系统

#### 5.3.2 即兴表演 🎭

**规则**:
1. 随机抽取表演主题
2. 随机决定表演顺序
3. 每人在60秒内表演/描述
4. 其他玩家评分 (1-10分)
5. 累加平均分

**界面**:
- 主题展示
- 表演者提示
- 计时器
- 评分转盘

#### 5.3.3 创意命名 🎨

**规则**:
1. 系统随机展示一个普通物品
2. 三人各自想一个创意名字
3. 所有人展示后投票选出最有趣的
4. 被投票最多的得30分

**界面**:
- 物品展示 (图片)
- 创意名字输入
- 匿名投票系统
- 结果展示

#### 5.3.4 神转折 📝

**规则**:
1. 系统给出故事开头和结尾
2. 三人共同填充故事中间部分
3. 每人负责一段
4. 最终生成完整故事
5. 评选最有趣的神转折

**界面**:
- 开头和结尾显示
- 中间段落编辑区
- 故事预览
- 评分投票

### 5.4 混合创新类 (D)

#### 5.4.1 命运转盘 🎲

**规则**:
1. 使用转盘随机抽取挑战类型
2. 该类型的挑战难度随机
3. 三人共同完成挑战
4. 根据表现获得分数
5. 转满5格结算

**界面**:
- 3D转盘动画
- 当前挑战显示
- 难度星级
- 总进度条

#### 5.4.2 限时任务 🏃

**规则**:
1. 系统同时发布多个小任务
2. 三人分工合作完成
3. 每个人只能完成自己负责的任务
4. 在60秒内完成尽可能多的任务
5. 每完成一个任务得10分

**界面**:
- 任务列表
- 任务分配显示
- 倒计时
- 完成状态追踪

#### 5.4.3 终极问答 🎯

**规则**:
1. 混合所有类型的综合挑战
2. 20道题目，随机抽取
3. 答对得分，答错不得分
4. 最终根据总分排名

**界面**:
- 题目展示
- 答案选项
- 进度条
- 实时排名

## 6. 游戏模式详细设计

### 6.1 自由模式 🎮

**特点**:
- 首页展示所有游戏类型
- 玩家可自由选择
- 支持单人练习和联机对战
- 独立计时计分

**流程**:
```
选择模式 → 选择游戏类型 → 设置 → 开始游戏 → 结算
```

**功能**:
- 游戏收藏功能
- 历史最高分记录
- 快速重玩

### 6.2 闯关模式 🏆

**特点**:
- 按顺序完成4种类型挑战
- 完成一个才能解锁下一个
- 最终综合评定等级

**等级系统**:
- 🥉 青铜 (0-200分)
- 🥈 白银 (200-400分)
- 🥇 黄金 (400-600分)
- 💎 钻石 (600-800分)
- 👑 王者 (800分以上)

**流程**:
```
开始闯关 → 类型1挑战 → 解锁 → 类型2挑战 → 解锁 → 类型3挑战 → 解锁 → 类型4挑战 → 最终评定
```

**功能**:
- 进度保存
- 失败可重试
- 时间记录

### 6.3 随机挑战 🎲

**特点**:
- 每轮随机抽取游戏
- 三人同时挑战同一题目
- 紧张刺激

**流程**:
```
开始挑战 → 随机抽取 → 公布 → 挑战 → 结果 → 下一轮 (共5轮) → 总结算
```

**功能**:
- 悬念揭晓动画
- 即时排名
- 连续胜场记录

### 6.4 循环赛 🔄

**特点**:
- 三人轮流进行各种游戏
- 积分累加制
- 最终排名

**流程**:
```
开始循环赛 → 游戏1 → 积分 → 游戏2 → 积分 → 游戏3 → 积分 → ... → 最终排名
```

**规则**:
- 每种游戏至少进行一轮
- 胜者得3分，平局各得1分，败者得0分
- 最终累加积分
