# 三人行 - 三人联机游戏平台

🎮 创新的三人联机游戏平台，提供多种游戏类型和模式，支持手机、平板和电脑。

## ✨ 功能特点

- 🎯 **4种游戏模式**：自由模式、闯关模式、随机挑战、循环赛
- 🎮 **6款创意游戏**：涵盖协作、竞技、创意、混合类型
- 📱 **响应式设计**：完美支持手机、平板、电脑
- 🔊 **音效系统**：沉浸式游戏体验（Web Audio API）
- 🌙 **深色模式**：保护眼睛，随时切换
- ♿ **无障碍支持**：键盘导航、ARIA标签、高对比度

## 🎮 游戏类型

### 协作挑战类
- 🎨 **你画我猜接龙** - 轮流画画，最后猜出主题
- 🧩 **协同解密** - 合作解开谜题

### 竞技对抗类
- 🧠 **知识抢答** - 快速抢答，考验知识面
- ⚡ **反应挑战** - 看谁反应最快

### 创意表达类
- ✍️ **故事接龙** - 轮流续写故事，打造独特剧本

### 混合创新类
- 🎲 **命运转盘** - 融合运气和技巧的综合性挑战
- 🎯 **记忆挑战** - 测试你的记忆力

## 🚀 快速开始

### 方法一：直接打开
直接在浏览器中打开 `index.html` 即可开始游戏！

### 方法二：本地服务器
```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js
npx serve .

# 使用 PHP
php -S localhost:8000
```

然后访问 `http://localhost:8000`

## 🎯 游戏模式说明

### 自由模式
自由选择想玩的游戏，适合休闲娱乐。

按顺序完成不同类型的挑战，考验综合实力。
